# Unity Registry Package Installation

Install Unity registry packages via MCP. OpenUPM packages (Unity-MCP, UniTask, Zenject) are handled by the onboarding script.

---

## Install via MCP (Unity registry packages)

These are Unity registry packages — install via the MCP `package-add` tool:

- `com.unity.inputsystem` (Input System)
- `com.unity.render-pipelines.universal` (URP)

## Verify

Tell the developer to switch to the Unity Editor and wait for package import to complete. Unity will recompile scripts automatically after detecting the new packages.
